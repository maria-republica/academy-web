Web Mobile Specialist

Inicio

2 de septiembre.

Objetivo

Durante el curso aprenderán a construir aplicaciones usando webpack e integrarlo con frameworks como React. Aprenderás como evolucionar tu sitio a una progressive web app y el uso de las Chrome Dev Tools para análisis y depuración de aplicaciones web.

Perfil del aspirante

Este curso va dirigido a personas que hayan construido aplicaciones web y estén interesados en aumentar sus habilidades para construir aplicaciones robustas siguiendo buenas prácticas. El aspirante debe dominar los siguientes tópicos en JS: Promesas, Contextos, Prototype, modificación de DOM y suscripción a eventos usando Vanilla JS

Al salir podrás:

Crear tu propio Workflow de desarrollo, construir aplicaciones usando componentes en React y resolver problemas asociados al aspecto técnico del ecosistema web (performance, accesibilidad), además de entender el background para crear progressive web apps.

Espectativa económica laboral después de completar el programa:
$35,000 - 60,000 pesos.*

Duración

El programa Web Mobile Specialist tiene duración de 1 mese y medio (90 horas).
Las clases son presenciales de Lunes a Viernes de 7:00PM a 10:00PM.

*El mínimo de alumnos para abrir un grupo será a partir de 8 participantes.
