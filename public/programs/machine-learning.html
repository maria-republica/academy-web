Machine Learning

La primera herramienta para atacar problemas relacionados con patrones de datos es Machine Learning, una disciplina de informática relacionada con el diseño de algoritmos que permite a las computadoras evolucionar comportamientos basados ​​en datos empíricos. Estos algoritmos se puede organizar en la siguiente jerarquía: aprendizaje supervisado, aprendizaje no supervisado y aprendizaje semi-supervisado. Por lo tanto, Machine Learning es más que nada una disciplina interdisciplinaria que se ocupa del descubrimiento de patrones en grandes conjuntos de datos que involucran métodos de inteligencia artificial, Machine Learning, estadísticas, optimización y sistemas de bases de datos. Hoy en día, esta intersección de disciplinas ha florecido en lo que muchas personas llaman Ciencias de datos, haciendo de Machine Learning una de sus piedras angulares.

Objetivo

Este es un curso teórico y práctico de 180 horas que presenta a los estudiantes los conceptos de Machine Learning, para procesar y analizar datos de diferentes fuentes. El énfasis está en varios problemas de Machine Learning, sus soluciones y su uso para el análisis de datos. Los estudiantes desarrollarán una comprensión del proceso de Machine Learning, además de aprender técnicas para Machine Learning y aplicarlas en la resolución de problemas diversos de Machine Learning.

Perfil del aspirante

El aspirante debe tener conocimiento en:

  • Álgebra Lineal
  • Probabilidad
  • Análisis de algoritmos

Al salir podrás:

Integrarte a equipos de trabajo para diseñar e implementar los algoritmos que se necesitan para el manejo masivo de data. Con estos análisis serás capaz de encontrar los patrones en la data que te permitirán hacer suposiciones acerca de ella y de esta manera hacer tests de hipótesis para responder preguntas acerca de la data.
