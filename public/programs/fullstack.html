Objetivo

El objetivo del programa es que aprendas a crear productos digitales o plataformas web de 0 a 100%. Durante el curso aprenderás a desarrollar tanto la parte Frontend (La parte visual) como Backend (La parte lógica)del proyecto. Todo esto te dotará del conocimiento necesario para adentrarte en la industria tech y convertirte en un gran desarrollador.

  • FRONTEND: La parte de la página con la que el usuario tiene contacto, vive en el navegador y abarca tanto el aspecto visual como interactivo.

  • BACKEND: La parte que maneja los datos, permisos e información. Por lo general suele vivir en el servidor. Aquí se maneja la base de datos, los distintos tipos de usuario y la lógica de negocio.


Para quién es este programa


  • Cambio de carrera: Buscas un cambio de carrera pero tienes nulos conocimientos técnicos. Tomarías el bloque de "intro a la programación para empezar a entender este mundo."

  • Skill up: Haz tomado clases online pero tienes muy poca experiencia laboral o buscas mejorar tus skills tecnológicos.



El programa está dividido en cuatro bloques estratégicos para que desarrolles los skills necesarios para tu cambio de carrera y formes parte de la industria tech:

Al salir serás capaz de:

  • Tener un perfil Fullstack JavaScript Jr.
  • Analizar problemáticas complejas y crear soluciones digitales.
  • Crear un modelo de datos, APIs y aplicaciones frontend bajo el conjunto de tecnologías usadas en el MERN stack (MongoDB, Express, React, Node).
  • Llevar proyectos pequeños por tu cuenta
  • Acoplarte a equipos de desarrollo complejos.

Temario

+ Github

Esencial para cualquier desarrollador, es una herramienta que nos apoyará a generar versiones para respaldar y ordenar nuestro código además de ayudarnos a trabajar en equipo de una manera cómoda.

  • Repositorios para proyectos
  • Commits y manejo de versiones
  • Branchs y trabajo en equipo
  • Gitflow

+ JavaScript

Es un lenguaje de programación que debido a su capacidad de adaptarse para diversos usos o entornos está siendo utilizado en la industria para el desarrollo de sus productos. Entenderlo a fondo te abrirá la posibilidad de utilizar diferentes frameworks y dejar de ver la programación como magia.

  • Paradigmas de programación en JS
  • Javascript (ES6 y posterior)
  • Asíncronia en JS

+ Frontend

Se refiere a la parte de la página que interactúa con el usuario, abarca desde características visuales, como colores hasta interacciones complejas. El frontend también se preocupa por que la página cargue de manera ágil y se adapte a los diferentes navegadores.

  • HTML Y CSS
  • CSS frameworks y prepocesadores
  • DOM (Document Object Model)
  • Javascript (ES6 y posterior)
  • Peticiones y asincronía
  • Optimización
  • React
  • Redux

+ Backend

Aprenderás a guardar información en una base de datos, crear usuarios y definir la lógica tu plataforma necesite para funcionar. El backend vive en los servidores, por que aprenderás a configurar uno y montar tu página.

  • Node JS
  • Express
  • MongoDB
  • Modelado de datos
  • Rest APIs
  • Servidores

+ Proyectos e implementaciones

Haremos varios proyectos que simularán el ambiente de trabajo de la vida real. Te servirán para ganar experiencia y armar un portafolio robusto para mostrar.

+ Soft Skills

Te preparamos con metodologías de organización, formas de pasar las entrevistas y todas las habilidades que necesitas para trabajar en equipo.
